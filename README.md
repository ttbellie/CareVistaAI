# CareVista AI

CareVista AI is an Android health tracking application that includes BMI calculation, period tracking, health logging, and a prototype AI assistant.

## Features
- BMI Calculator
- Period Tracker
- Health Tracker
- AI Assistant (Prototype)

## Technology
- Android (Java)
- XML UI
- SharedPreferences
- Rule-based logic

## Notes
This project is a prototype. AI responses are simulated using predefined logic.
